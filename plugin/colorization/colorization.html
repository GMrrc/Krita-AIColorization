Plugin Krita - Colorisation Automatique des Dessins

Ce plugin permet de faire des inférences sur le canvas dans Krita. En utilisant une intelligence artificielle, il colorise automatiquement les dessins à partir d'un fichier de référence pour les couleurs et du canvas actuel.

Fonctionnalités principales

  • Colorisation automatique de croquis avec une image de référence.
  • Prend en charge les dessins de style animé.
  • Basé sur le modèle Anime-Sketch-Colorizer.
  • Utilisation de réseaux de neurones pour améliorer la précision des couleurs.

Installation des dépendances

Si vous rencontrez des problèmes avec les dépendances, veuillez installer les paquets nécessaires à partir du fichier requirements.txt inclus dans le plugin.

Voici les principales bibliothèques requises :

pytorch
torchvision
numpy
opencv-python
matplotlib

Référence GitHub

Le plugin utilise le code et le modèle fournis par le dépôt GitHub suivant :

Anime-Sketch-Colorizer - GitHub Repository

Données et Modèle

Le modèle est pré-entraîné pour coloriser des dessins de style animé. Il utilise le dataset suivant :

Anime Sketch Colorization Pair - Kaggle Dataset

Vous pouvez télécharger le modèle pré-entraîné à partir de ce lien :

Télécharger le modèle pré-entraîné (449MB)

Paramètres d'entraînement

Paramètre Valeur
Learning rate 2e-4
Batch size 2
Epoch 25
Optimiseur Adam
Data Augmentation RandomResizedCrop(256), RandomHorizontalFlip()

Test et Utilisation

Pour tester le modèle ou entraîner votre propre modèle, veuillez vous référer aux fichiers train.ipynb et test.ipynb fournis dans le dépôt GitHub.
